## リリースノート

Klaytn v1.7.1 では、トランザクションの処理中に accountKey フィールドが変更される可能性のある重大なメモリ破壊のバグが修正されています。

**NOTE**：すべてのノードオペレータ（CN/PN/ENオペレータ）は、ノードバイナリをこのバージョンに更新する必要があります。

## 修正

- 適用中のトランザクションで、Txメモリの破壊が発生する問題を修正(#1060)。
