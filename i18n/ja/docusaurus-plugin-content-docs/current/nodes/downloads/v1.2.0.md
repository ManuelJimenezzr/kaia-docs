## Klaytn v1.2.0のリリースノート

## ハイライト - サービス・チェーン初の公式リリース<a id="highlight-the-first-official-release-of-service-chain"></a>

このバージョンで正式にサービスチェーンをサポートすることを発表できることを嬉しく思います。
サービス・チェーンは、水平方向のスケーラビリティを実現するために設計されたKlaytnのレイヤー2ソリューションである。 サービスチェーンは、クレイトンのメインチェーンから独立して運営され、独自のガバナンスを持つことができる。 サービス・チェーンについて詳しくは、[スケーリング・ソリューション](../../learn/scaling-solutions.md)をご覧ください。

### 新機能<a id="new-features"></a>

- サービス・チェーンの特徴
- データのアンカリング：サービスチェーンのデータは `ChainDataAnchoring` トランザクションによって親チェーンに固定することができる。
- チェーン間の価値移転：KLAYとKCTは、サービスチェーンと親チェーンの間で移転することができる。
- 高可用性：偶発的なハードウェア/ソフトウェア障害によるダウンタイムを低減するため、複数のブリッジノードをH/A構成でセットアップできます。
- サービス・チェーンのための新しいAPI
- klay_getDecodedAnchoringTransactionByHash
- subbridge_getParentOperatorBalance
- subbridge_getChildOperatorBalance
- サブブリッジ_getAnchoringTxHashByBlockNumber
- subbridge_getValueTransferOperatorThreshold
- subbridge_setValueTransferOperatorThreshold
- subbridge_registerOperator
- subbridge_getRegisteredOperators

### 修正<a id="fixes"></a>

- txpoolの競合状態を修正。

### 改善点<a id="improvements"></a>

- アンロックされたトークンのみがチェーンを越えて移動できる。
