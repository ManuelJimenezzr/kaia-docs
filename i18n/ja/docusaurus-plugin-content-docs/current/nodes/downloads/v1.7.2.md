## リリースノート

Klaytn v1.7.2では、CNがトランザクションの処理とブロックの生成を同時に行った場合にパニックを引き起こす可能性のあるレースコンディションの問題が修正された。 このパッチはCN固有の問題にのみ適用される。 その他のノードタイプ（EN、PN）は、Klaytnクライアントをこのバージョンに更新する必要はありません。

## 修正

- txpoolにおける競合状態の問題を修正 (#1071)
