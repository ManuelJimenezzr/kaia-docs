## Klaytn v1.5.1のリリースノート

### 修正<a id='fixes'></a>

- 不良ブロックの問題を修正 (#576)
  - 状態トライの `dereference` ロジックの競合状態を解決した。
  - トライ・ノード・キャッシュの `コミット` および `参照` 中に `dereference` が発生しないようにした。

### その他<a id='miscellaneous'></a>

- RC RPMの公開 / PRコメントへのRCバイナリのリンク (#581)
  - RCになるとRPMを公開する仕事を追加した。 RC版へのインストールやアップグレードは以前よりずっと簡単になりました。
  - 今後は、各リリースPRのコメントに記載されているリンクからRC版バイナリをダウンロードできるようになります。
- Golangのバージョンをv1.14.6にアップグレード (#582)
  - goコマンド、コンパイラ、リンカ、vet、データベース/sql、エンコーディング/json、net/http、reflect、testingパッケージの修正が含まれる。 (https\://golang.org/doc/devel/release.html)
