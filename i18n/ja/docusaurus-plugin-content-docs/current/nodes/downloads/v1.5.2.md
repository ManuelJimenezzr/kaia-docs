## Klaytn v1.5.2のリリースノート

### 新機能

- ブロックチェーンのデータをリアルタイムでRDBにプッシュするチェーンデータフェッチャーを実装。
  - トークン転送履歴をロードするためのチェーンデータフェッチャを実装しました(#586)。
  - 内部/復帰トランザクションデータをロードするためのチェーンデータフェッチャを実装 (#587)
  - 契約データをロードするチェーンデータフェッチャーを実装 (#588)
- サービスチェーンノードでKASアンカーAPIをサポートしました。
  - サービスチェーン内のブロックを、コマンドオプションを使ってKAS Anchor API経由で簡単にBaobab/Cypressにアンカーできるようになりました(#653)
  - JSON-RPC経由でのアンカートランザクション送信を許可する`txpool.allow-local-anchortx`コマンドオプション (#614)
  - 新しいアンカーデータ型（JSON形式）のサポート (#614)

### その他

- 新しいデータベース実装（DynamoDB）を追加（#633, #624, #615, #598）。
- genesisブロックを既存の状態で初期化するための`overwrite-genesis`フラグを追加した(#649)。
- stateDBキャッシュインターフェイスを実装し、さまざまなタイプのキャッシュ実装をサポートしました(#646)。
