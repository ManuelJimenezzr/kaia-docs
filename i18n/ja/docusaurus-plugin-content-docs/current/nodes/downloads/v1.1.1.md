## Klaytn v1.1.1のリリースノート

#### 修正<a id="fixes"></a>

- ジニ係数の計算のバグを修正。

#### 改善点<a id="improvements"></a>

- テストケースの追加
- パッケージの説明を更新
- サービスチェーンブリッジ契約に著作権メッセージを追加。
- 未使用コードの削除

## Klaytn v1.1.0のリリースノート

### 新機能<a id="new-features"></a>

- 特定のブロック番号でスマートコントラクトのメソッドを呼び出すことをサポートします。
- ガバナンス協議会メンバーの複数ステーキング契約をサポート

#### 修正<a id="fixes"></a>

- 様々なバグを修正

#### 改善点<a id="improvements"></a>

- コンセンサス・メッセージは、ラウンド・ナンバーを同期させるためにガバナンス・カウンシルのメンバーに伝えられる。
- パッケージのリファクタリング（報酬、ガバナンスなど）
- doc.goのパッケージ説明を更新
- ユニットテストの追加
