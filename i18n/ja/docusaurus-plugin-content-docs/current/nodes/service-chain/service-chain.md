# サービスチェーン

## 対象読者<a id="intended-audience"></a>

- 高いTPS、最小限の取引手数料、データプライバシーを必要とするdApp開発者。
- ローカル・プライベート・ネットワークや元帳データベースをテスト用に構築したい人。

## サービス・チェーンの概要<a id="service-chain-overview"></a>

カイア・サービスチェーンには次のような特徴がある。

- 即座の最終決定。
- クロスチェーン・トークン転送。
- メインチェーンへのデータ固定。
- 企業のセキュリティ要件を満たすマルチシグブリッジ契約。
