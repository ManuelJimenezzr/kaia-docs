# 単位

単位変換のために、Kaia-SDKは`kaia`、`Gkei`、`kei`の変換ユーティルをサポートしている。 対応ユニット](https://docs.kaia.io/learn/kaia-native-token/#units-of-kaia-)を参照。

<CH.Spotlight>
  <CH.Code>
    ```java UnitConvertExample.java
    package org.web3j.example.utils;

    import org.web3j.utils.KaiaConvert;

    public class UnitConvertExample {
        public static void main(String[] args) {
            System.out.println("Convert from kei to kaia " + KaiaConvert.fromKei("100000000000000", KaiaConvert.Unit.KAIA));
            System.out.println("Convert from Gkei to kaia " + KaiaConvert.fromKei("1000000000", KaiaConvert.Unit.GKEI));

            System.out.println("Convert from kaia to kei " + KaiaConvert.toKei("1", KaiaConvert.Unit.KAIA));
            System.out.println("Convert kaia to kei " + KaiaConvert.toKei("1", KaiaConvert.Unit.GKEI));
        }.
    }
    ```

    ---

    ```zsh output
    ❯ java UnitConvertExample.java
    keiからkaiaに変換 1
    Gkeiからkaiaに変換 1
    kaiaからkeiに変換 100000000000000000000
    ```
  </CH.Code>

  ---

  org.web3j.utils.KaiaConvert\*\* パッケージから `KaiaConvert` をインポートする。


  ---

  他の単位をKaiaに変換するには、`KaiaConvert.fromKei`を使う。 最初のパラメータは値、2番目のパラメータは `kei/Gkei/kaia` の単位型である。
