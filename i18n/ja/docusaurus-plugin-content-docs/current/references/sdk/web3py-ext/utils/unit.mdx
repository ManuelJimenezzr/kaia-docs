# 単位

単位変換のために、Kaia-SDKは`kaia`、`Gkei`、`kei`の変換ユーティルをサポートしている。 対応ユニット](https://docs.kaia.io/learn/kaia-native-token/#units-of-kaia-)を参照。

<CH.Spotlight>
  <CH.Code>
    ```py unit_utils.py
    from web3py_ext import extend
    from web3 import Web3

    def main()：
        # (kei/Gkei/kaia)からkeiに変換
        print("KaiaからKeiへ", Web3.to_kei(1, 'Gkei'))

        # keiから(kei/Gkei/kaia)への変換
        print('KeiからKaiaへ',Web3.from_kei(1000000000000000000,'kaia'))

    main()
    ```

    ---

    ```zsh output
    ❯ unit_utils.py
    カイアからケイへ 1
    ケイからカイアへ 1
    ```
  </CH.Code>

  ---

  web3をkaia web3に拡張するために、**web3py_ext**から**extend**をインポートする。

  ```py unit_utils.py focus=1
  ```

  ---

  (Gkei/kaia)`から`kei`に変換するには、`Web3.to_kei(value, unit)\`を使う。

  <CH.Code>
    ```py unit_utils.py focus=5:6
    ```

    ---

    ```zsh output
    ❯ unit_utils.py
    カイアからケイへ 1
    ```
  </CH.Code>

  ---

  kei`から`(kaia/Gkei)`に変換するには、`Web3.from_kei(value, unit)\`を使う。

  <CH.Code>
    ```py unit_utils.py focus=8:9
    ```

    ---

    ```zsh output
    ❯ unit_utils.py
    ケイからカイアへ 1
    ```
  </CH.Code>
</CH.Spotlight>
