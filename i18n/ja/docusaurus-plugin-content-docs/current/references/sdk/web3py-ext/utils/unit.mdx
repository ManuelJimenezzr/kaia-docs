# 単位

単位変換のために、Kaia-SDKは`kaia`、`Gkei`、`kei`の変換ユーティルをサポートしている。 対応ユニット](https://docs.kaia.io/learn/kaia-native-token/#units-of-kaia-)を参照。

<CH.Spotlight>
  <CH.Code>
    ```py unit_utils.py
    from web3py_ext import extend
    from web3 import Web3

    def main():
        # convert from (kei/Gkei/kaia) to kei
        print("From Kaia to Kei", Web3.to_kei(1, 'Gkei'))

        # convert from kei to (kei/Gkei/kaia)
        print('From Kei to Kaia',Web3.from_kei(1000000000000000000,'kaia'))

    main()
    ```

    ---

    ```zsh output
    ❯ py unit_utils.py
    From Kaia to Kei 1
    From Kei to Kaia 1
    ```
  </CH.Code>

  ---

  web3をkaia web3に拡張するために、**web3py_ext**から**extend**をインポートする。

  ```py unit_utils.py focus=1
  ```

  ---

  (Gkei/kaia)`から`kei`に変換するには、`Web3.to_kei(value, unit)\`を使う。

  <CH.Code>
    ```py unit_utils.py focus=5:6
    ```

    ---

    ```zsh output
    ❯ py unit_utils.py
    From Kaia to Kei 1
    ```
  </CH.Code>

  ---

  `kei`から`(kaia/Gkei)`に変換するには、`Web3.from_kei(value, unit)`を使う。

  <CH.Code>
    ```py unit_utils.py focus=8:9
    ```

    ---

    ```zsh output
    ❯ py unit_utils.py
    From Kei to Kaia 1
    ```
  </CH.Code>
</CH.Spotlight>
