- 更新軟件包說明

### 新功能<a id="new-features"></a>

- 支持在特定區塊編號上調用智能合約方法
- 支持為管理委員會成員簽訂多個盯人合同

#### 修復<a id="fixes"></a>

- 修復了多個錯誤

#### 改進<a id="improvements"></a>

- 向治理委員會成員傳播共識信息，以同步整數
- 重構軟件包（獎勵、管理等）
- 更新了 doc.go 中的軟件包說明
- 添加了更多單元測試
