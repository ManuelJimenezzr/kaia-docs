# caver.rpc

caver.rpc "是一个提供与 kaia Node 的 rpc 调用相关的功能的软件包。

## 类别<a id="class"></a>

### RPC<a id="rpc"></a>

```javascript
caver.rpc
```

RPC "是一个包含[Klay]、[Net]和[Governance]的类别。

**属性**

| 名称   | 类型     | 描述                                 |
| ---- | ------ | ---------------------------------- |
| klay | [Klay] | [Klay] 提供带有`klay`名称空间的JSON-RPC 调用。 |
| net  | [Net]  | [Net] 提供带有`net`名称空间的JSON-RPC 调用。   |
| 治理   | \[治理]  | \[治理] 提供带有`治理`名称空间的JSON-RPC 调用。    |
