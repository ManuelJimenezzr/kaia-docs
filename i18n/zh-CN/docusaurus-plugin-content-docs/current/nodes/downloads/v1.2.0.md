## Klaytn v1.2.0 发布说明

## 亮点 - 服务链首次正式发布<a id="highlight-the-first-official-release-of-service-chain"></a>

我们很高兴地宣布，该版本正式支持服务链。
服务链是 Klaytn 的第二层解决方案，旨在实现横向可扩展性。 服务连锁店独立于 Klaytn 主连锁店，可以有自己的管理机构。 要了解有关服务链的更多信息，请访问 [Scaling Solutions]（.../.../learn/scaling-solutions.md）。

### 新功能<a id="new-features"></a>

- 服务链功能
- 数据锚定：服务链数据可通过 "ChainDataAnchoring "事务锚定到父链。
- 跨链价值转移：KLAY 和 KCT 可在服务链和母链之间转移。
- 高可用性：可在 H/A 配置中设置多个网桥节点，以减少因硬件/软件意外故障而造成的停机时间。
- 服务链的新应用程序接口
- klay_getDecodedAnchoringTransactionByHash
- subbridge_getParentOperatorBalance
- subbridge_getChildOperatorBalance
- subbridge_getAnchoringTxHashByBlockNumber
- subbridge_getValueTransferOperatorThreshold
- subbridge_setValueTransferOperatorThreshold
- 子桥注册操作员
- subbridge_getRegisteredOperators

### 修复<a id="fixes"></a>

- 修正了 txpool 中的一个竞赛条件。

### 改进<a id="improvements"></a>

- 只有解锁的代币才能跨链移动。
