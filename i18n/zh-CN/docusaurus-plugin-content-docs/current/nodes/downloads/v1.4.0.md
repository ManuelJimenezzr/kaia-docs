## Klaytn v1.4.0 发布说明

### 新功能<a id="new-features"></a>

- 密钥存储 V4
  - Klaytn 采用了新的密钥存储标准 [KIP-3](https://klaytn.github.io/kips/KIPs/kip-3)，以支持其账户。

### 修复<a id='fixes'></a>

- 修复了更新 `stakingInfo` 时因时间问题导致的间歇性坏块现象。
- 修正了在执行 `debug.traceBlock` 时因 API 与三节点垃圾收集之间的时间问题而产生的恐慌。
- 修正了区块同步过程中的一个错误报告。

### 改进<a id='improvements'></a>

- 优化的存储系统
  - Klaytn 现在会计算每个区块的存储根哈希值，而不是每个交易的存储根哈希值。
  - Klaytn 同时计算三角形哈希根。
- 增加状态三缓存大小
  - 增加默认缓存大小，以提高性能。
- 獾数据库版本升级至 v1.6.0。
- 替换度量包
- 已添加软件包说明
- 提高测试代码覆盖率
