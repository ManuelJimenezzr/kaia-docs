## Klaytn v1.5.2 发布说明

### 新功能

- 实施链数据获取器，实时向 RDB 推送区块链数据
  - 实现了链式数据获取器，以加载令牌传输历史记录 (#586)
  - 实现了链式数据获取器，以加载内部/还原事务数据 (#587)
  - 实施链式数据获取器以加载合同数据 (#588)
- 在服务链节点中实现了对 KAS 锚点 API 的支持（#591）
  - 可使用命令选项通过 KAS 锚点 API 将服务链中的块轻松锚定到 Baobab/Cypress 上 (#653)
  - txpool.allow-local-anchortx "命令选项允许通过 JSON-RPC 提交锚点事务 (#614)
  - 支持新的锚点数据类型（JSON 格式） (#614)

### 杂项

- 添加了新的数据库实现（DynamoDB）（＃633、＃624、＃615、＃598）
- 添加了 "overwrite-genesis "标记，以使用现有状态初始化创世块（#649）
- 实现了 stateDB 缓存接口，以支持各种类型的缓存实现 (#646)
