## Klaytn v1.5.1 发布说明

### 修复<a id='fixes'></a>

- 修正了一个坏块问题（#576）
  - 解决了状态三 "引用 "逻辑的竞赛条件问题。
  - 防止在 "提交 "和 "引用 "三节点缓存时 "取消引用"。

### 杂项<a id='miscellaneous'></a>

- RC RPM 发布/RC 二进制链接至 PR 注释 (#581)
  - 在涉及 RC 时，为发布 RPM 添加了一项工作。 您可以比以前更轻松地安装/升级到 RC 版本。
  - 从现在起，您可以从每个发布 PR 的注释中描述的链接下载 RC 版本的二进制文件。
- 将 Golang 版本升级至 v1.14.6 (#582)
  - 包括对 go 命令、编译器、链接器、vet 以及数据库/sql、编码/json、net/http、reflect 和测试软件包的修复。 (https://golang.org/doc/devel/release.html)
