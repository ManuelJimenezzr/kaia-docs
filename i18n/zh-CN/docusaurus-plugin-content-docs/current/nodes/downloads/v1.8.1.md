## 发布说明

该版本包含一个增强 MetaMask 兼容性的修复程序。 它会影响块头 API 的返回格式。 因此，建议您将使用`eth_getHeader`相关 API 的 EN 更新到该版本。

### 修复

- 区块头 API 仅在 "EthTxType "硬分叉后返回 "baseFeePerGas "字段 (#1247)
